# 🚀 30 Days DSA Challenge

Welcome to my **30 Days DSA Challenge** repository!

This repository documents my journey of mastering the most important Data Structures and Algorithms (DSA) patterns through consistent daily problem-solving.

The goal of this challenge is not just to solve problems, but to understand the logic, recognize patterns, and build strong problem-solving skills for coding interviews.

---

# 🎯 Challenge Goal

- Build strong DSA fundamentals.
- Improve problem-solving and pattern recognition.
- Stay consistent by solving problems every day.
- Prepare for Software Engineer internship and placement interviews.
- Share my learning journey publicly through GitHub and LinkedIn.

---

# 📅 Challenge Plan

For every DSA pattern, I'll spend **5 days** mastering it:

- ✅ Day 1–2: Easy Problems
- ✅ Day 3–4: Medium Problems
- ✅ Day 5: Hard Problem

This structured roadmap helps me build a strong foundation before moving on to the next pattern.
